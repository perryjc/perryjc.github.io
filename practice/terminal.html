how to veiw html code

  1. press CMD+shift+C on your computer on the page you want to veiw
  2. navigate to the "elements" tag
  3. alternatively right click and select "veiw page source" witch will open the index.html file (see below image)

Here are some common text tags:

  • Bold text
  • Italic text
  • Underlined text
  • Superscripttext
  • Subscripttext
  • Strike through
  • Highlighted text
  • Quoted text

How to make your first HTML file

  1. open your terminal
  2. cd to your website directory
  3. enter "touch index.html" this will make a index.html file if it doesnt already exist
  4. Now type "vim index.html"
  5. press "i" to start editing
  6. copy the first 2 lines of this file and put them in your new file
  7. add a new head (remember to close it!)
  8. in the head make a "title" element
  9. in the "title" element write "hello!"
  10. close the element
  11. then add a body (remember to close it!)
  12. in the body use a "p" element and type your name
  13. close the "p" element
  14. on the last line close an html tag
  15. press esc and then type ":wq" this will save and exit the file
  16. now run index.html with google!
